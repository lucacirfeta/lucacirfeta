### 🚀 Featured projects

**[gravi-signal-ml](https://github.com/lucacirfeta/dante-gravi-signal-ml)** — Research side project
Unsupervised anomaly detection on LIGO/Virgo gravitational-wave data (O4a). Uses DINOv2 embeddings + UMAP/HDBSCAN clustering to surface novel glitch classes not covered by Gravity Spy, cross-validated across detectors (H1/L1).
`Python` `PyTorch` `DINOv2` `UMAP` `HDBSCAN`

**[re-wine]** — Personal project
An Android app to manage a personal wine cellar, with barcode scanning via the Open Food Facts API. Built to track and rediscover Italian regional wines (Barolo, Barbaresco, Nebbiolo).
`Flutter` `Riverpod` `SQLite` `GoRouter`
